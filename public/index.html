🔥 Discovery & Trending
🔥 Trending Manga Working
GET/api/v1/trending?period=week&limit=10
Manga/manhwa trending berdasarkan bookmark dalam periode waktu. Period: day, week, month, all.
⭐ Popular Content Working
GET/api/v1/popular?metric=bookmarks&limit=10
Konten populer berdasarkan metric. Metric: bookmarks, views, ratings, avg_rating.
🆕 Latest Manga Working
GET/api/v1/latest?limit=20
Manga/manhwa terbaru yang ditambahkan ke platform, diurutkan berdasarkan tanggal tambah.
📖 Manga / Komik
📚 Semua Manga Working
GET/api/v1/mangas?page=1&limit=20&type=manga
Daftar semua manga/manhwa/manhua dengan filter type, status, genre, tags, sortir rating, views, dll.
🔎 Detail Manga Working
GET/api/v1/mangas/:slug
Detail lengkap manga berdasarkan slug. Termasuk daftar chapter, rating, tags, dan metadata.
➕ Tambah Manga Admin
POST/api/v1/mangas
Tambah manga baru ke platform. Butuh autentikasi role admin/moderator.
Butuh Bearer Token (admin)
⭐ Rating Manga Working
POST/api/v1/mangas/:id/rate
Beri rating 1-10 untuk manga. Satu user satu rating, bisa diupdate.
Butuh Bearer Token
📄 Chapter List Working
GET/api/v1/mangas/:id/chapters
Daftar semua chapter untuk manga tertentu. Diurutkan berdasarkan nomor chapter.
📄 Chapters & Baca
📖 Baca Chapter Working
GET/api/v1/chapters/:slug
Gambar-gambar chapter untuk dibaca. Mengembalikan array URL gambar / server mirror.
⬅️ Navigasi Chapter Working
GET/api/v1/chapters/:slug/navigation
Navigasi prev/next chapter untuk manga. Memudahkan implementasi tombol lanjut/kembali.
➕ Tambah Chapter Admin
POST/api/v1/chapters
Tambah chapter baru dengan gambar-gambar. Mendukung upload multi-file.
Butuh Bearer Token (admin)
🎬 Donghua API
🏠 Home Donghua Working
GET/api/v1/donghua/home
Data beranda Donghua: latest_release (20 episode terbaru) + completed_donghua (20 series tamat).
▶️ Ongoing Donghua Working
GET/api/v1/donghua/ongoing?page=1&limit=20
Daftar donghua yang sedang tayang (status ongoing) dengan pagination.
✅ Completed Donghua Working
GET/api/v1/donghua/completed?page=1&limit=20
Daftar donghua yang sudah selesai/tamat dengan pagination.
🔍 Pencarian Donghua Working
GET/api/v1/donghua/search?q=battle+through
Cari donghua berdasarkan judul menggunakan full-text search MongoDB dengan ranking relevance.
🎞️ Detail Donghua Working
GET/api/v1/donghua/:slug
Detail lengkap donghua: synopsis, genre, seasons, daftar episode, rating, studio, dan navigasi.
🍿 Nonton Episode Working
GET/api/v1/donghua/episode/:episodeSlug
Data streaming episode donghua. Termasuk navigasi prev/next episode dan info series.
🏷️ Daftar Genre Working
GET/api/v1/donghua/genres
Semua genre donghua yang tersedia beserta slug untuk digunakan di filter.
🗂️ By Genre Working
GET/api/v1/donghua/genre/:genre?page=1
Donghua berdasarkan genre tertentu (slug atau nama). Contoh: /genre/action, /genre/fantasy.
📅 By Season / Tahun Working
GET/api/v1/donghua/year/:year?page=1
Donghua berdasarkan tahun rilis season. Contoh: /year/2024, /year/2023.
🎬 Episodes & Seasons (Animasi)
🗓️ Daftar Season Working
GET/api/v1/mangas/:id/seasons
Semua season untuk satu series animasi, diurutkan berdasarkan nomor season.
📺 Daftar Episode Working
GET/api/v1/mangas/:id/episodes?seasonId=
Semua episode untuk series animasi. Optional filter ?seasonId= untuk season tertentu.
▶️ Detail Episode Working
GET/api/v1/episodes/:id
Detail episode animasi termasuk URL video/streaming, navigasi prev/next, dan info series.
🔍 Search
🔍 Global Search Working
GET/api/v1/search?q=naruto&type=manga
Pencarian global dengan 3 method fallback: full-text → regex → fallback. Filter type: manga, manhwa, manhua, donghua.
👤 Auth & User
📝 Register Working
POST/api/v1/auth/register
Daftarkan akun baru. Body: username, email, password. Kirim email verifikasi otomatis.
🔑 Login Working
POST/api/v1/auth/login
Login dan dapatkan accessToken + refreshToken. Access token berlaku 15 menit.
🔄 Refresh Token Working
POST/api/v1/auth/refresh
Perbarui access token menggunakan refresh token. Tidak perlu login ulang.
👤 Profil Saya Working
GET/api/v1/users/me
Data profil user yang sedang login. Termasuk statistik bookmark, reading history, dsb.
Butuh Bearer Token
🌐 Public User Profile Working
GET/api/v1/users/:userId/public
Profil publik user lain: username, bio, avatar, stats. Data sensitif disembunyikan.
🔖 Bookmarks & History
🔖 Bookmark Saya Working
GET/api/v1/bookmarks?page=1&limit=20
Daftar manga yang di-bookmark user. Support filter status dan pagination.
Butuh Bearer Token
➕ Toggle Bookmark Working
POST/api/v1/bookmarks/:mangaId
Toggle bookmark manga — tambah jika belum ada, hapus jika sudah ada. Auto-invalidate trending cache.
Butuh Bearer Token
📜 Reading History Working
GET/api/v1/histories?page=1&limit=20
Riwayat chapter yang sudah dibaca user, diurutkan dari yang paling baru.
Butuh Bearer Token
📁 Collections
📁 Koleksi Saya Working
GET/api/v1/collections?page=1&limit=20
Daftar koleksi yang dibuat user. Bisa private atau public.
Butuh Bearer Token
📂 Buat Koleksi Working
POST/api/v1/collections
Buat koleksi baru. Body: name, description, visibility (private/public).
Butuh Bearer Token
📊 Trending Collections Working
GET/api/v1/collections/public/trending
Koleksi publik paling populer berdasarkan jumlah item dan view count.
💬 Comments & Reviews
💬 Komentar Manga Working
GET/api/v1/comments/manga/:mangaId?page=1
Daftar komentar untuk manga tertentu. Support pagination dan sortir.
✍️ Tulis Komentar Working
POST/api/v1/comments
Tambah komentar baru untuk manga. Mendukung reply (nested comment).
Butuh Bearer Token
📝 Reviews Manga Working
GET/api/v1/reviews/manga/:mangaId?page=1
Ulasan panjang (review) untuk manga tertentu. Berbeda dari komentar singkat.
🏷️ Tags
🏷️ Daftar Tag Working
GET/api/v1/tags?page=1&limit=50
Semua tag yang tersedia di platform. Bisa digunakan untuk filter manga.
